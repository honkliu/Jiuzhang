# 管理员配置

管理员帮助只说明配置方法。聊天、札记、家谱、票夹、啪等功能的日常使用，请看用户帮助。

## 访问配置

入口在顶部导航的 **配置**。保存后配置立即生效，不需要重启 API。

配置规则：

- 增加一行表示启用该配置。
- 删除一行表示取消该配置。
- 修改后点击当前配置块右上角的 **保存**。
- 不确定当前状态时，先点 **刷新** 重新读取服务器配置。

## kankan@kankan

`kankan@kankan` 是根管理员，可以维护全局访问配置。

配置项：

- **让不同域用户可互相看见**：填写来源域和可见域。来源域用户可以看到可见域的家谱。
- **家谱域**：填写允许使用家谱功能的邮箱域，例如 `shaol.com`。
- **管理员**：填写系统管理员邮箱。管理员天然可以管理自己邮箱域下已启用的家谱域。
- **家谱管理人**：填写用户邮箱和管理域。该用户可以在指定域创建、导入、修改家谱。

检查项：

- **域可见性预览** 用来确认跨域可见规则是否符合预期。
- **家谱管理人** 可以是普通用户；不需要先加入管理员列表。
- 邮箱显示为黄色背景或棕色文字时，表示配置引用的用户还不存在。

## 普通管理员

普通管理员只能维护自己有权管理域内的家谱管理人。

配置方法：

- 在 **家谱管理人** 中添加用户邮箱和管理域。
- 管理域必须是当前管理员已经能编辑的域。
- 保存后查看 **每个家谱谁能创建和查看**，确认域配置和家谱数量。
- 查看 **用户能管理和编辑的家谱**，确认用户最终可编辑域。

## 常见配置顺序

新开一个家谱域时：

1. 用 `kankan@kankan` 添加 **家谱域**。
2. 如需跨域查看，添加 **让不同域用户可互相看见**。
3. 添加负责该域的 **管理员** 或 **家谱管理人**。
4. 保存后刷新，检查预览表。

给普通用户授权管理家谱时：

1. 在 **家谱管理人** 增加用户邮箱。
2. 填写要授权的管理域。
3. 保存后检查用户实际可编辑域。

## 故障检查

- 用户看不到家谱入口：确认他的邮箱域已加入 **家谱域**。
- 用户不能创建或修改家谱：确认他在 **管理员** 或 **家谱管理人** 配置中拥有对应域。
- 跨域看不到家谱：检查 **让不同域用户可互相看见** 和 **域可见性预览**。
- 保存后结果不对：点击 **刷新**，确认服务器返回的配置是否已更新。
